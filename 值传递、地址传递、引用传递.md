### 1.值传递

```c
void Increment(int value)
{
    value++;
}
int main()
{
    int a = 5;
    Increment(a);
    LOG(a);//输出结果:5
}
```

### 2.地址传递

```c
void Increment(int* value)
{
    (*value)++;
}
int main()
{
    int a = 5;
    Increment(&a);
    LOG(a);//输出结果:6
}
```

### 3.引用传递

```c
void Increment(int &value)
{
    //引用相当于是原变量的别名，对 value++ 实际就是对 a++。
	//没有创建新副本，直接修改原始变量。
    value++;
}
int main()
{
    int a = 5;
    Increment(a);
    LOG(a);//输出结果:6
}
```
