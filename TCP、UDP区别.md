| 特性           | TCP                       | UDP                     |
| -------------- | ------------------------- | ----------------------- |
| 面向连接       | 是（需握手）              | 否（直接发）            |
| 可靠性         | 高，丢包重传，顺序保证    | 低，不重传，不保证顺序  |
| 速度           | 较慢（复杂）              | 快（轻量）              |
| 是否有拥塞控制 | 有                        | 无                      |
| 数据单位       | 字节流                    | 数据报（报文段）        |
| 应用场景       | Web、文件传输、聊天系统等 | 视频、音频、DNS、游戏等 |
